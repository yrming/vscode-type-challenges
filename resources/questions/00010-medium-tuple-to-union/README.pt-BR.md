> Traduzido pelo Google, abra um PR para ajudar a melhorar a tradução.

Implemente um `TupleToUnion<T>` genérico que cubra os valores de uma tupla até sua união de valores.

Por exemplo

```ts
type Arr = ['1', '2', '3']

type Test = TupleToUnion<Arr> // expected to be '1' | '2' | '3'
```
