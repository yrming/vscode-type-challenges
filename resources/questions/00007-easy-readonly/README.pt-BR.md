> Traduzido pelo Google, abra um PR para ajudar a melhorar a tradução.

Implemente o genérico `Readonly<T>` integrado sem usá-lo.

Constrói um tipo com todas as propriedades de T definidas como somente leitura, o que significa que as propriedades do tipo construído não podem ser reatribuídas.

Por exemplo:

```ts
interface Todo {
  title: string
  description: string
}

const todo: MyReadonly<Todo> = {
  title: "Hey",
  description: "foobar"
}

todo.title = "Hello" // Error: cannot reassign a readonly property
todo.description = "barFoo" // Error: cannot reassign a readonly property
```
